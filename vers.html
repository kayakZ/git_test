Главное правило использования Git - никогда не производи отладку в Мастер-ветви. Мастер - ветвь содержит только отлаженный код. Мастер-ветвь аккумулирует в себе все дополнения других программистов! Если требуется интеграция вашего кода с кодом других программистов, то Вам необходимо копировать код из Мастера в вашу рабочую ветвь, выполнить в ней отладку, а затем залить в мастер!
